Задача: Для квадратной матрицы

Найти максимальное число, которое заканчивается на цифру 5 и расположено под
побочной диагональю. Если такое число не может быть найдено, на экран ничего
выводить не нужно, функция main должна вернуть код ошибки.

Проверяем вывод ошибок при неправильном вводе размера матрицы:

Значения не типа int:
in: a 2
out: error

in: 2 a
out: error

in: .2 2
out: error

in: 2 .2
out: error

Значения вне диапазона [1, 10]
in: -1 2
out: error

in: 2 -1
out: error

in: 2 12
out: error

in: 12 2
out: error

Матрица не является квадратной
in: 4 3
out: error

В матрице 1 столбец и одна строка (такое число не может быть найдено)
in: 1 1
out: error

Проверяем вывод ошибок при неправильном вводе элементов матрицы:

Значения не типа int:
in: 2 2
.2
out: error

in: 2 2
a
out: error

такое число не может быть найдено
in: 3 3
0 100 1
-1 -2 9
13 14 16
out: error

Проверяем правильность работы программы:
Обычные ситуации:

Проверяем, что искомый элемент ищется исключительно под побочной диагональю:
in: 3 3
125 -3 -4
0 555 1
-1 3 5
out: 
5

В матрице всего один элемент под побочной диагональю, оканчивающийся цифрой 5
in: 3 3
1 4 0
2 2 5
1 1 20
out: 
5

В матрице несколько элементов под побочной диагональю, оканчивающихся цифрой 5
in: 3 3
1 4 0
-2 2 5
1 1 25
out: 
25

Проверкана правильность работы с отрицательными числами
in: 4 4
1 4 0 13
-2 2 14 5
1 1 25 -555
0 -5 -25 35
out: 
35
