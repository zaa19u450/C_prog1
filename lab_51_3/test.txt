ОБЩИЕ ОШИБОЧНЫЕ СИТУАЦИИ
Неправильные переданные аргументы или их количество

Недостаточно аргументов
in: app.exe
out: error

in: app.exe test.bin
out: error

in: app.exe с test.bin
out: error

Слишком много аргументов
in: app.exe s t y t bin.bin test.txt
out: error

Неизвестная команда
in: app.exe t test.bin
out: error

Небинарный файл
in: app.exe s test.txt
out: error



ОШИБКИ ДЛЯ ЗАПОЛНЕНИЯ ФАЙЛА
Количество чисел для заполнения файла - не целое положительное число
in: app.exe c -10 test.bin
out: error

in: app.exe c 0 test.bin
out: error

in: app.exe c .8 test.bin
out: error

in: app.exe c d test.bin
out: error

РАБОЧИЕ СИТУАЦИИ ДЛЯ ЗАПОЛНЕНИЯ ФАЙЛА
