Задача: Преобразовать исходную матрицу следующим образом:
вставить строку из чисел -1 перед каждой строкой, в которой есть хотя бы два элемента,
сумма цифр которых нечетна;

Проверяем вывод ошибок при неправильном вводе размера матрицы:

Значения не типа int:
in: a 2
out: error

in: 2 a
out: error

in: .2 2
out: error

in: 2 .2
out: error

Значения вне диапазона [1, 10]
in: -1 2
out: error

in: 2 -1
out: error

in: 2 12
out: error

in: 12 2
out: error

Если вводится всего один столбец, то такая ситуация считается ошибочной,
так как ни в одной строке не найдется 2 элементов с нечетной суммой цифр
in: 3 1
out: error

Проверяем вывод ошибок при неправильном вводе элементов матрицы:

Значения не типа int:
in: 2 2
.2
out: error

in: 2 2
a
out: error

Проверяем правильность работы программы:
Обычные ситуации:

В матрице всего 1 строка (где есть 2 элемента с нечетной суммой цифр)
in: 1 4
1 0 13 10
out: 
-1 -1 -1 -1
1 0 13 10

В матрице во всех строках есть 2 элемента с нечетной суммой цифр
in: 3 3
11 23 45
1 7 9
13 14 16
out: 
-1 -1 -1
11 23 45
-1 -1 -1
1 7 9
-1 -1 -1
13 14 16

В матрице ни в одной строке нет 2 элементов с нечетной суммой цифр
in: 3 3
11 22 44
1 10 91
0 0 13
out: 
11 22 44
1 10 91
0 0 13

В матрице в некоторых строках есть 2 элемента с нечетной суммой цифр
(+ проверка на корректность обработки отрицательных чисел и нуля)
in: 3 3
-11 -23 45
0 -22 -4
13 -14 16
out: 
-1 -1 -1
-11 -23 45
0 -21 -4
-1 -1 -1
13 -14 16
