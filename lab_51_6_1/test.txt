Проверяем ошибочные ситуации:

пустая последовательность:
in: p
out: error
in: .5
out: error

одно число в последовательности:
in: 1 p
out: error

Все элементы последовательности равны
in: 3 3 3 3 3 3 3 p
out: error


Обработка рабочих ситуаций
вся последовательность убывает/возрастает
in:  1 2 3 4 5 p
out: 5
in:   4 3 2 1 0 w
out: 5

В последовательности есть и возрастание, и убывание
in:  1 4 6 6 3 3 2 1 q
out: 3

В последовательности есть и возрастание, и убывание и их длины равны
in:  1 2 3 2 1 2 3 r
out: 3

Работа с отрицательными и двузначными числами:
in: -100 -50 0 10 20 -10 -20 -30 -40 s
out: 5

В последовательности есть невозрастание/неубывание 
in:  1 2 2 2 3 4 5 5 5 4 4 k
out: 4

Все элементы последовательности равны
in: 3 3 3 3 3 3 3 p
out: 0
